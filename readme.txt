== Installation ==

1. Go to `Plugins` in the Admin menu
2. Click on the button `Add new`
3. Search for `Plum Island Media Telemetry` and click 'Install Now' or click on the `upload` link to upload `plum-island-media-telemetry.zip`
4. Click on `Activate plugin`
